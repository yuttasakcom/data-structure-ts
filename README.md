## Recursion
```
แนวทางการเขียนโปรแกรมแนวทางหนึ่ง ที่มีแนวคิดมาจากคณิตศาสตร์ ภาษาไทยเรียก "การทำซ้ำ" โดยลักษณะ "การทำซ้ำ" จะเป็นการเรียก(call)ฟังก์ชั่นตัวเองให้ทำงานซ้ำอีกครั้งหนึ่ง แต่ยังคงค้าง(suspend)การประมวลผลก่อนหน้าที่ยังไม่ได้คำนวณไว้ใน memory เมื่อถึงจุดที่ฟังก์ชั่นคำนวณได้ผลลัพธ์แล้ว ก็จะส่งผลลัพธ์ย้อนกลับมาทำส่วนที่ค้างไว้ใน memory และทำการประมวลผลไปเรื่อยๆจนกว่าจะได้ผลลัพธ์
จะสังเกตุได้ว่า Recursion จะเริ่มด้วยข้อมูลที่มีขนาดใหญ่ และเมื่อเรียกตัวเอง ข้อมูลจะเล็กลงเรื่อยๆ มีลักษณะเหมือนตุ๊กตาหุ่นที่ซ้อนกันของญี่ปุ่น
```
